# Tarot

Frontend-only, TypeScript React tarot reading app.

## Run

```bash
npm install
npm run dev
```

## Notes

- No backend, API endpoint, database, or server route is used.
- Tarot cards, zodiac signs, spread positions, and reading texts live in `src/data`.
- Readings are saved locally with `localStorage`.
- The UI uses Tailwind CSS with a boutique mystical palette.
